Badge Configuration Examples

Here are some example JSON configurations you can use in your product meta fields:

Simple Badge (Text Only)

[
  {
    "text": "New"
  }
]

Badge with Custom Colors

[
  {
    "text": "Premium",
    "background_color": "#FFD700",
    "text_color": "#000000",
    "border_color": "#FFA500"
  }
]
